# Charlemagne-AI
Charlemagne is my AI project to both Learn and Produce a Fully working AI using C++ with prototyping in Python to build an AI that uses both recursive and predictive Knowledge bases.

# My Direction
1. This project will use PUBLIC DOMAIN Resources to train this AI on Understanding Humanity in a way that conventional AI have missed.
2. The main test is to see if an AI trained to have a form of humanity in it compared to standard guardrails is safe and possible
3. The end goal it for me to create an AI that is better for the environment compared to data centers that harm environments
4. I want to make something that can advance research, understanding of the world, and general advancments.
5. The main inspiration is Rasputin from the video-game Destiny.
6. Charlemagne will have 12 smaller job dedicated AI named and based off of the 12 paladins of Charlemagne.
7. I will prototype this AI in python to get a understanding of the direction, sizing, and code.
8. The finished Project will be coded in C++ as to reduce the size and cut unnecessary bloat.
